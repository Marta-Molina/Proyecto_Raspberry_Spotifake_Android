# AppMusic

Aplicación Android desarrollada como proyecto para Programación multimedia y dispositivos móviles.  
Esta app ofrece funcionalidades similares a las de otras aplicaciones con similar propósito, permitiendo a los usuarios crear listas de reproducción, añadir canciones, artistas y álbumes, así como gestionar su contenido musical.

---

## Descripción

La aplicación permite a los usuarios disfrutar de una experiencia musical personalizada y completa. Cuenta con una interfaz intuitiva y moderna, dividida en varias funcionalidades clave:

- **Listas de reproducción:** Los usuarios pueden crear y gestionar listas de reproducción.
- **Añadir canciones, artistas y álbumes:** Permite la búsqueda y adición de canciones, artistas y álbumes a las listas.
- **Reproductor de música:** Permite reproducir canciones directamente desde las listas.
- **Login y acceso a datos mediante API:** Se implementará un sistema de login para acceso a datos personales y la integración con APIs externas para gestionar contenido musical.

---

## Tecnologías y herramientas utilizadas

- **Lenguaje:** Kotlin  
- **IDE:** Android Studio  
- **SDK:** Android SDK (mínimo API 24)  
- **Bibliotecas:** Glide
- **Layouts:** ConstraintLayout, RecyclerView, CardView
- **Control de versiones:** Git + GitHub
